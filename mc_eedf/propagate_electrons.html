propagate_electrons¶

class src.propagate_electrons.Electron¶

An simulated electron that may be propagated in a simulated plasma. An Electron() class instance will keep track of it’s own position, velocity, acceleration, and energy.

src.propagate_electrons.run_electrons()¶

When completed, this function will return a list of electron class instances. Each instance has accessible lists of their respective positions, velocities, accelerations, and energies experienced over a corresponding time. The run_electrons function below takes 2 arguments.

Returns

list of electorn class instances

Return type

electrons
